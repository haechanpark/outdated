CSE Projects

Scratch Project

Early September 2017- Mid September 2017

Objective: Create a functioning game or story
Principles of Software Engineering
Project Type: Peer Project (Pairs)
For the Scratch project, my partner and I were tasked to make a game using the Scratch engine. Besides this, the door to creative freedom was open. In return, our group decided to work on an RPG, or a role-playing game, that allows the player to progress through worlds, level up, and fight monsters. Many bugs started springing up, but nearly all were resolved by simply looking at the code. This took the most time in the project. In the meantime, our group also had to complete an online notebook, which contained logs and general information on how the project was made and the progression of creation. The criteria for grading the project was located on an instruction document with an attached rubric on the same document. The end result/grade was a relatively high for our group, as we managed to accomplish much work in the limited time frame. Through this two-week project, I learned the basics of variables and functions as well as learned how to efficiently use the Scratch engine.

Image Manipulation Project

Early March 2018- Mid March 2018

Objective: Create functions that can alter an image. Client: Product
Principles of Software Engineering
Project Type: Peer Project (Pairs)
For the Image Manipulation Project, our group had to make functions that can alter an image in order to satisfy our client. Our client was a company that was trying to advertise a product. This company was Pear technologies (Fictional). For us, this was one of the most difficult projects in the CSE class as we ran into issue upon issue. In order to fix the the issues, we went through every line of code to look for incorrect code. Most of the bugs were actually just caused by incorrect method usage and therefore were solved on the most part. The project also had an online notebook to track our data and changes for the images that we were manipulating (four images). In my opinion, I think our project satisfied the client well and in general worked decently at the end. Throughout this project, I learned many concepts, mainly the skill of debugging and the ability to properly image manipulate using the python code.
